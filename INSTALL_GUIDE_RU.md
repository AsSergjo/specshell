# Руководство по установке и удалению

Это руководство поможет вам установить или удалить расширение контекстного меню для создания спектрограмм.

**ВАЖНО:** Все действия, описанные ниже, необходимо выполнять с правами **администратора**.

---

## Установка расширения

Установка добавит в контекстное меню Проводника Windows новый пункт (например, "Создать спектрограмму"), который будет появляться при нажатии правой кнопкой мыши на аудиофайлах.

**Шаги для установки:**

1.  Убедитесь, что файл `SpectrogramContextMenu.dll` находится в той же папке, что и файл `install_ext.bat`. Этот файл (`.dll`) появляется после сборки проекта.
2.  Нажмите правой кнопкой мыши на файл `install_ext.bat`.
3.  В появившемся меню выберите пункт **"Запуск от имени администратора"**.
4.  Если система запросит разрешение на внесение изменений, нажмите "Да".
5.  Вы должны увидеть системное окно с сообщением об успешной регистрации библиотеки. Нажмите "ОК".
6.  В окне консоли появится сообщение `Successfully registered the extension.`.
7.  **Для того чтобы изменения вступили в силу, скрипт предложит перезапустить Проводник Windows (`explorer.exe`). Нажмите 'Y' (Да) или 'N' (Нет). Перезапуск приведет к кратковременному исчезновению рабочего стола и панели задач — это нормально.**

После этого расширение будет установлено и готово к работе.

---

## Удаление расширения

Удаление полностью убирает пункт из контекстного меню и отменяет регистрацию библиотеки в системе.

**Шаги для удаления:**

1.  Нажмите правой кнопкой мыши на файл `uninstall_ext.bat`.
2.  В появившемся меню выберите пункт **"Запуск от имени администратора"**.
3.  Если система запросит разрешение на внесение изменений, нажмите "Да".
4.  Вы должны увидеть системное окно с сообщением об успешном удалении регистрации библиотеки. Нажмите "ОК".
5.  В окне консоли появится сообщение `Successfully unregistered the extension.`.
6.  **Для полного удаления расширения из интерфейса, скрипт предложит перезапустить Проводник Windows (`explorer.exe`). Нажмите 'Y' (Да) или 'N' (Нет). Это действие безопасно.**

После этого расширение будет полностью удалено из вашей системы.
